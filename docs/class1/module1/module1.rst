Module 1 - Enterprise Features
==============================

In this section, we will investigate two features identified as necessary for "Enterprise Readiness" -- Access Management and Cluster Resiliency.

Lab 1 will show how to add and manage an external Active Directory authentication provider.
Lab 2 will demonstrate how to join additional NGINX Controller instances in a cluster to provide instance resiliency.
